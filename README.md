# Asciiarter

Modern, minimal Image → ASCII dashboard with a clean glassmorphism UI.

## Features

- **Dashboard UI**: Wide, modern dark/light themes with glass panels
- **Color Picker**: User-selectable accent/glow colors
- **Fast Output**: Colored preview rendered with **canvas** (avoids huge DOM trees)
- **Controls**: Width (chars), charset presets + custom charset, invert mapping
- **Actions**: Copy, Download `.txt`, Export `.png`, Regenerate, Clear
- **Drag & Drop + Upload**: JPG/PNG/GIF/WebP

## Technical Details

- React + Vite
- Tailwind CSS (v4) + shadcn-style UI primitives
- Canvas-based image sampling + optional canvas color preview
